📋 RÉSUMÉ EXÉCUTIF
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 VISION
   Permettre aux serviteurs de Dieu de préparer leurs messages plus efficacement
   en combinant recherche IA dans les Écritures et organisation de fiches.

👥 PUBLIC CIBLE
   Pasteurs et prédicateurs francophones

💡 PROBLÈME RÉSOLU
   ✗ Recherche manuelle longue dans plusieurs versions de la Bible
   ✗ Difficulté à trouver des enseignements connexes (Branham, pasteur)
   ✗ Notes de prédication désorganisées
   ✗ Préparation de messages chronophage

✅ SOLUTION
   ✓ Recherche IA sémantique (mots-clés + Q&A)
   ✓ Sources multiples (3 Bibles + Branham + Pasteur)
   ✓ Fiches de prédication structurées
   ✓ Sauvegarde et organisation automatique

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 FONCTIONNALITÉS MVP (7 semaines)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 🔐 AUTHENTIFICATION
   • Inscription/Connexion (Email + Google OAuth)
   • Reset password
   • Protection des routes

2. 🔍 RECHERCHE IA
   • Mode Recherche (mots-clés, thèmes, versets)
   • Mode Q&A (questions en langage naturel)
   • Sources: Darby, Segond, Martin, Branham, Pasteur
   • Résultats en <3 secondes
   • Historique des recherches

3. 📝 FICHES DE PRÉDICATION
   • CRUD complet (Create, Read, Update, Delete)
   • Structure: Titre, Thème, Versets, Plan, Notes
   • Ajout de versets depuis recherche
   • Sauvegarde automatique
   • Liste et recherche de fiches

4. 📊 DASHBOARD
   • Fiches récentes (5 dernières)
   • Recherches récentes (5 dernières)
   • Statistiques d'utilisation
   • Accès rapide aux fonctionnalités

5. ⚙️ ADMINISTRATION
   • Upload prédications (.txt, .docx, .pptx)
   • Indexation automatique (Inngest)
   • Gestion des documents
   • Statut d'indexation

6. 💎 FREEMIUM
   • Plan Gratuit: 10 recherches/jour, 5 fiches max
   • Plan Premium: 9.99€/mois, illimité
   • Intégration Stripe (paiements + webhooks)
   • Gestion des abonnements

1. Créer le repo GitHub
   $ cd /home/hp/logos-rv
   $ git init
   $ git add .
   $ git commit -m "docs: complete project documentation"
   $ gh repo create logos-rv --public --source=. --push

2. Initialiser Next.js
   $ npx create-next-app@latest . --typescript --tailwind --app --src-dir

3. Setup Prisma
   $ npm install @prisma/client prisma
   $ npx prisma init
   $ # Copier prisma-schema.prisma → prisma/schema.prisma
   $ npx prisma generate
   $ npx prisma migrate dev --name init

4. Configurer les services externes
   • Neon (PostgreSQL)
   • Upstash (Redis + Vector)
   • OpenAI API
   • Google OAuth
   • Stripe

5. Commencer le développement
   $ npm run dev
